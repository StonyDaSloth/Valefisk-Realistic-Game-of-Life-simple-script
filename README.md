## commands
```-c [usd amount]``` - *converts usd to pound*

```-con [usd amount]``` - *converts usd to pound*

```-convert [usd amount]``` - *converts usd to pound*

**Example Usage**

> -c 500
> > (result) : 395

> -con 500
> > (result) : 395

>  -convert 500
> > (result) : 395
